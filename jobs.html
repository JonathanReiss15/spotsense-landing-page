Open Source Project Intern

Job Title: Open Source Project Intern

Reports to: Jonathan Reiss, CTO of SpotSense

Job Description: With this learning-focused opportunity, interns will be working closely with the founders and technical team to learn how to build open source web-based projects for the iOS developer community. These projects will be released under an MIT license. At the same time, interns will be able to learn the inner mechanics of an early stage startup and how technical and business requirements intersect.

Responsibilities:

  • Design, test, and develop open source (MIT) projects
  • Attend standup meetings each week
  • Learn!

Interns will learn:

  • How an early-stage startup technical team operates
  • How to deploy + maintain projects on Amazon Web Services
  • How to build + ship software for the open source community
  • How to collaborate on open source projects with industry and community partners
  • The SpotSense technical stack (React + DynamoDB + API Gateway + Lambda Functions)
  • How to build software tools for other developers

Applicants should have an understanding of:

  • Web languages, e.g. HTML, CSS, Javascript
  • Past experience with a frontend framework is a plus (React.js or Vue)
  • Node.js
  • Rest APIs

Important "soft" skills:

  • Ability to work in a team
  • Able to solve problems independently
  • Strong research (aka StackOverflow) skills
  • Thirst for learning
  • Able to communicate well in writing or speech

Compensation: Unpaid
